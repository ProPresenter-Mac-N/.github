## Overview

ProPresenter Advanced is a live presentation application from Renewed Vision designed for worship services, live events, conferences, and broadcast production with multi-screen output, media playback, and real-time content switching that handles the demanding live production environment. It manages lyrics, slides, video, and lower thirds across multiple output screens simultaneously with the reliability that live production requires.

Multi-screen output sends different content to different screens simultaneously — lyrics to the congregation display, notes to the stage display, and video to the broadcast output — handling the multi-destination output that live production environments require. Each output is independently controlled with its own content, transitions, and formatting.

Lyrics presentation displays song lyrics with real-time triggering, enabling operators to advance through songs with the worship team rather than pre-timing. The lyrics system handles verse/chorus structure, chord charts for musicians, and automatic formatting that adapts text to different screen sizes and aspect ratios.

Media integration plays video, audio, images, and web content alongside slide-based presentations with layer-based compositing that combines multiple media sources on screen. The media engine handles the format variety that live production encounters.

For worship leaders, event producers, and broadcast teams who need reliable live presentation with multi-screen output and real-time content control, ProPresenter Advanced is the live presentation platform that handles the complexity of multi-destination production.

## Performance

ProPresenter balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive.

Apple Silicon acceleration is fully utilized on M-series chips. On Intel machines, hardware acceleration is used wherever available.
